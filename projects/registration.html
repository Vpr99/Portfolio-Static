Course planning is critically important for students, but registration tools often feel like they're stuck in 1995. In this concept, I reworked my college's tool to be more usable and powerful by providing some valuable context.

Scope

Research, Web

Tools

Sketch, Photoshop

Dashboard

Unlike Facebook or Instagram, where using the application is the end-goal, this tool is judged by how little time users have to spend completing tasks. Surfacing links for frequently used tools at the top of the page will reduce the amount of searching required to use common functions.

Search

Returning contextual and relevant search results allows students to make snap judgements and build their schedules more quickly: "Do I need to take this course for my major?" or "Is the class full?"

Class Detail Page

Eligibility for a class is the first deciding factor for students and should be shown accordingly. Next is often how well the class fits in the rest of their schedule. A week preview gives a holistic view of all saved classes, reducing the cognitive load necessary to mentally build their schedule.
